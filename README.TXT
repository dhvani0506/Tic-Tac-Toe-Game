PROJECT TITLE: TicTacToe Game

PURPOSE OF PROJECT: This is where you will be able to play the TicTacToe game and play against the CPU.

VERSION or DATE: 4/28/25

HOW TO START THIS PROJECT: Select void Main of Main class to run the game.

AUTHORS: Dhvani & Brianna

USER INSTRUCTIONS:

1. To play TicTacToe, you will need to select the Main class to start the game. Then, you can type in your name and enter, to start.

2. Now, the main objective of this game is to make three shapes in a row horizontally, vertically, or diagonally to win.

3. The Player, which will be you, will play as X, where you will need to play the X shapes in a row.

4. Your Opponent, the CPU, will play as O, where you'll need to carefully stategize their move to block them if they get 2 in a row.
   Each of you will take a turn one at a time to place your marks. 

5. If you're able to successfully fill in three in a row, you'll gain a point.

6. However, if the CPU wins, they'll end up getting the point instead of the Player.

7. In an instance where the board ends up getting filled, neither the Player nor the CPU will gain a point as you both draw.

8. By the end of the game, whoever has the most points will be the winner of TicTacToe.
